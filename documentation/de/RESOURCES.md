# RESOURCES (DE, verschoben)

Assets & Konventionen (Texturnamen, Modelle, Sprachdateien) aus ursprünglicher Datei.

## Namensschema
Präfix `1of90_` für Texturen/Sounds/Modelle.

## Qualitätscheck
Dateinamenspräfix, Größe ≤256px, korrekte Transparenz, Lizenz.

## Typische Fehler
Pink/Schwarz → Pfad
Verzerrtes Modell → Ursprung/Skalierung
Flackern Transparenz → Blend aktivieren

## Roadmap
Asset-Validierungs-Skript, Spriteatlas, dynamische Farbvarianten.
